  ESP32 DevKit V1 — Pin Mapping v2.2
  ----------------------------------------------------------------------------

  INPUTS:
    GPIO  4  ← Mode+ button (handlebar, pullup)
    GPIO  5  ← Mode- button (handlebar, pullup)
    GPIO  0  ← Encoder BTN (BOOT pin, internal pullup)
    GPIO 16  ← Encoder A (CLK, interrupt)
    GPIO 17  ← Encoder B (DT)
    GPIO 18  ← Ignition pulse coil (interrupt, optocoupler)
    GPIO 19  ← Oil pressure switch (LOW = pressure OK)
    GPIO 34  ← ADC: Thermistor (NTC 10kΩ)
    GPIO 35  ← ADC: Battery voltage divider
    GPIO 36  ← ADC: Stator voltage divider (v2.0+)

  OUTPUTS:
    GPIO 25  → PWM: Exhaust valve servo
    GPIO 26  → PWM: Airbox flap servo
    GPIO 27  → CDI Map A select (active LOW)
    GPIO 33  → CDI Map B select (active LOW)
    GPIO 32  → WS2812 RGB LED data

  I2C Bus (SDA=GPIO21, SCL=GPIO22):
    SSD1306 OLED @ 0x3C
    AS5600 Exhaust @ 0x36 (production: DRV8833)
    AS5600 Airbox   @ 0x37 (production: DRV8833)

  DRV8833 Production Motor Driver (USE_PRODUCTION_MOTOR=1):
    GPIO 14  → DRV8833 AIN1 (Exhaust valve)
    GPIO 13  → DRV8833 AIN2 (Exhaust valve)
    GPIO 23  → DRV8833 BIN1 (Airbox flap)
    GPIO  2  → DRV8833 BIN2 (Airbox flap)

  DEEP SLEEP WAKE SOURCES:
    GPIO  4  ← EXT0: Mode+ button (wake on LOW)
    GPIO 18  ← EXT1: Ignition pulse (wake on HIGH)

      │       └── ESP32 VIN (5V input)
      │
      ├── Voltage Divider #1 (Battery ADC)
      │       R1 = 100kΩ → Battery+
      │       R2 = 33kΩ  → GND
      │       Junction → GPIO 35
      │
      └── Voltage Divider #2 (Stator ADC)
              R1 = 100kΩ → Regulator output+
              R2 = 33kΩ  → GND
              Junction → GPIO 36

================================================================================
  v2.2 NEW FEATURES
================================================================================

  1. AUTO-RPM VALVE: Exhaust & airbox follow RPM curves per mode
     - 5 interpolation points per mode (RPM → valve position %)
     - Overrides static valve_percent when ENABLE_AUTO_RPM_VALVE=1

  2. FUEL ESTIMATION: mL/100km per mode + range + low fuel alert
     - RPM-adjusted consumption (idle -30%, high RPM +25%)
     - 16L tank, 3.4L reserve warning, 1.5L critical

  3. GEAR DETECTION: RPM/speed correlation estimates current gear
     - Uses NX650 gear ratios: 2.846, 1.857, 1.389, 1.091, 0.913
     - 15% tolerance, confidence percentage display

  4. DEEP SLEEP: 5 min engine off → ESP32 deep sleep (~10µA)
     - Wake sources: GPIO4 (Mode+ button) and GPIO18 (Ignition pulse)

  5. CONFIG MODE: Long press encoder (3s) → on-bike settings
     - Brightness → Valve Cal → Airbox Cal → Mode Select → Exit
     - Auto-exit after 10s inactivity
