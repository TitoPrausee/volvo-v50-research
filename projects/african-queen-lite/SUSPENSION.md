## 🔧 Installations-Hinweise

### Gabel-Upgrade (2-3h, Mittel)
1. Gabel aus Einbaum holen, Öl ablassen
2. Dämpfungsstange modifizieren (Löcher vergrößern) oder entfernen
3. Race Tech Emulator FEGV S4101 einsetzen
4. Wirth 0.55kg/mm Federn einbauen
5. Motul 5W Öl: 485ml pro Gabelhol, Standrohr 120-130mm von oben
6. **Kritisch:** Ölstelle genau messen! Zu viel Öl = hart, zu wenig = Blubbern

### Heck Mono-Conversion (3-4h, Schwer)
1. OEM Twin Shocks ausbauen
2. Mono-Bracket an Rahmen schweißen lassen ( professionelle Schweißerei!)
3. YSS Z-366 auf Bracket montieren
4. Federbasis und Zugstufe einstellen
5. **Kritisch:** Bracket muss exakt ausgerichtet sein! Schweißerei ~50-80€

### Bremsen (1-2h, Leicht-Mittel)
1. EBC FA185HH Pads einsetzen (direkter Tausch)
2. Venhill SS Leitung installieren (entlüften nicht vergessen!)
3. ATE TYP200 DOT4 einfüllen, entlüften
4. **Kritisch:** Bremsleitung korrekt routed, keine Knicke! Entlüftung sorgfältig durchführen.

### Reifen (1-2h, Mittel)
1. Alte Reifen+Schläuche demontieren
2. HD Schläuche in neue Mitas E-07 einsetzen
3. Aufziehen, auswuchten
4. Reifendruck: Front 1.9 bar, Rear 2.1 bar (Solo), 2.3 bar (mit Gepäck)
5. **Tipp:** Reifenmontage beim Händler (~15-30€ pro Reifen) lohnt sich für richtige Wuchtung
